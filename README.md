# Структура проєкту

Проєкт складається з таких основних файлів:
- **`subtitles.py`** – робота із субтитрами у форматі `.srt`
- **`search.py`** – пошук спільних сленгових виразів у двох датасетах
- **`main.py`** – обробка словника та генерація HTML-файлу
- **`index.html`** – інтерактивний веб-словник

## `subtitles.py`

  Цей скрипт відповідає за роботу з файлами субтитрів у форматі `.srt`. У ньому реалізовано функцію завантаження субтитрів із перевіркою на існування файлу. Якщо файл не знайдено, програма повертає помилку.
Основна функція цього модуля – паралельне виведення англійських та українських субтитрів, що дозволяє порівнювати оригінальні діалоги з перекладом.
**Основні функції:**
- `load_subtitles(file_path)` – завантаження файлів субтитрів
- `print_parallel_subtitles(english_subs, ukrainian_subs)` – паралельний вивід субтитрів

## `search.py`
  Цей скрипт використовується для порівняння двох наборів даних, що містять сленгові вирази з різних серіалів.
**Програма:**
- завантажує два CSV-файли
- виділяє сленгові вирази з кожного набору
- знаходить спільні вирази за допомогою перетину множин
- виводить детальну інформацію про кожен збіг
  Для кожного спільного виразу виводиться: український переклад, англійський та український контекст, а також кількість варіантів у кожному серіалі. Також формується новий CSV-файл із результатами порівняння.
Цей модуль використовується для порівняльного лексичного аналізу двох джерел сленгу.

## `main.py`
  Це основний скрипт проєкту, який відповідає за обробку словника та генерацію інтерактивного веб-додатку.
**У процесі роботи виконується кілька етапів:**
1. Зчитування CSV-файлу зі словником сленгу.
2. Очищення та нормалізація текстових даних.
3. Фільтрація за дозволеними сферами та емоційними характеристиками.
4. Обробка альтернативних перекладів і контекстів.
5. Формування структурованого набору даних.
6. Генерація HTML-файлу з інтерактивним інтерфейсом.
  Також визначаються список сфер вживання, список емоційних характеристик та кількість слів з альтернативними перекладами. Результатом є файл `index.html`.

## `index.html`
  Цей файл автоматично створюється скриптом `main.py` і є фінальним веб-інтерфейсом проєкту. Словник дозволяє користувачу:
- шукати сленгові вирази англійською або українською мовою
- фільтрувати дані за сферою вживання
- фільтрувати за емоційним забарвленням
- переглядати значення та контекст використання
- переглядати альтернативні переклади з прикладами
- використовувати автодоповнення при пошуку

**Кожен запис містить:**
- англійський сленговий вираз
- український відповідник
- значення
- сферу вживання
- емоційне забарвлення
- контекст англійською та українською мовами
  Якщо слово має альтернативні переклади, воно відкривається у вигляді модального вікна з додатковими прикладами.

## Результати
У результаті виконання проєкту:
- створено структурований словник молодіжного сленгу
- виявлено спільні сленгові вирази у двох серіалах
- збережено контекст використання мовних одиниць
- реалізовано інтерактивний веб-додаток для роботи зі словником
